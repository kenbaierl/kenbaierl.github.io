Control System PID Wind Turbine

The objective of this project was to understand the control system and make a block diagram of the wind turbine with disturbances and errors to account for while also finding the control system equation. Next, we need to find what kind of controller we need to use, be it a P, PI, PD, or PID controller to be able to account for all of the disturbances and errors in the project. We then had to find the design specifications including percent overshoot, settling time, rise time, steady-state error, and sensitivity by using the overall equation of the system. To find all of the design specifications and check the feedback and see which controller actually worked with the wind turbine, we used MATLAB to make visuals of the design specifications.
