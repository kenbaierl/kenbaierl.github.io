PCB Printed Turning Robot

This PCB-printed turning robot connects a battery pack to a switch that powers an Arduino and a voltage regulator which allows a constant 5V to a servo motor, ultrasonic sensor, and the Arduino. The Arduino connects to an H-bridge which allows the Arduino IDE code to turn the robot when an object passes in front of the ultrasonic sensor. One challenge that I faced during this project was that when the PCB was initially printed, the machine did not fully cut out some of the pins and Via holes and I had to solder together extra wires to make the circuit run.
