PCB Printed Turning Robot

This final project for a PCB making and printing class at Chapman University required designing a schematic using provided parts to create a robot that moves forward until an Arduino ultrasonic sensor detects an obstacle. Upon detection, the robot reverses and scans for nearby objects, turning away from any detected on the sides. We were provided with the Arduino code and parts, but had to create a schematic in Eagle, translate it to a board file ensuring no airwires remained, print the PCB, and solder the components to ensure the robot functioned correctly with the Arduino.
